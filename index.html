HTML Headings

HTML headings are titles or subtitles that you want to display on a webpage. <h1> defines the most important heading. <h6> defines the least important heading.

Think of HTML as a book. A book has one title and several chapters. The <h1> tag can be thought of as the title of the book, and the <h2>, <h3>, <h4>, <h5>, and <h6> tags can be thought of as the chapters of the book.

HTML Paragraphs

The HTML <p> element defines a paragraph.

A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph. Think of the HTML content as if it were a book with paragraphs where appropriate.

HTML Break Tag

The HTML <br> tag is like pressing Enter or Return on a keyboard. It moves the text to a new line without making a space like a paragraph does.

HTML Lists

HTML lists help organize related things on a webpage. They make it easy to show items in a clear and neat way, like a shopping list or a list of your favorite Blookets.

HTML Lists - Unordered

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

Unordered list items will be marked with bullets (small black circles) by default. 

          
Code Added to Code Editor (i.e., Codepen)

            <ul>
              <li>Grade 6</li>
              <li>Grade 7</li>
              <li>Grade 8</li>
            </ul>
 

Browser Output (What the User Will See)

  • Grade 6
  • Grade 7
  • Grade 8

HTML Lists - Ordered

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

Ordered list items will be marked with numbers by default.

          
Code Added to Code Editor (i.e., Codepen)

            <ol>
              <li>Deceptive Dinos</li>
              <li>Crypto Hack</li>
              <li>Gold Quest</li>
            </ol>
 

Browser Output (What the User Will See)

  1. Deceptive Dinos
  2. Crypto Hack
  3. Gold Quest

HTML Links

HTML links are hyperlinks. You can click on a link and go to another webpage. You can have that new webpage to open in the same browser window or in a new tab.

HTML Regular Links

The HTML <a> tag defines a hyperlink.

The href tells the browser where the link will go when you click it. The link text is the part you can see and click. When you click the link text, it takes you to a new webpage.

<a href="https://browning.edu">The Browning School</a>

In this example, the href tells the browser to go to the link https://browning.edu.

If you click on the link https://browning.edu it will open in the same browser tab/window.
